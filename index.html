This repository contains the source code for the Retail Insights Hub Dash application, which provides various business insights through interactive visualizations.

To run the app locally, follow these steps:

  1. Clone the repository: git clone https://github.com/thamiwinner/mega-mart-dashboard-.git
  2. Navigate to the project directory: cd mega-mart-dashboard-
  3. Install the required packages: pip install -r requirements.txt
  4. Run the Dash app: python app.py

The app will be available at http://127.0.0.1:8050/
